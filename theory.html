Bareiss-Montante Method

A short theoretical guide for solving systems of linear equations.

Calculator Theory

What is the Bareiss-Montante method?

The Bareiss-Montante method is an elimination method used to solve systems of linear equations through an augmented matrix. It transforms the matrix step by step until the coefficient part becomes diagonal.

Augmented matrix

A system with n variables is represented using an augmented matrix with n rows and n + 1 columns. The last column contains the constants from the right side of the equations.

n variables → n rows and n + 1 columns

Main formula

At each stage, the method uses a pivot and the previous pivot to calculate the new values of the matrix.

a'ij = (pivot · aij - aik · akj) / previous pivot

Step-by-step procedure

  1. Select the pivot from the diagonal.
  2. Keep the pivot row unchanged.
  3. Turn the other values in the pivot column into zero.
  4. Calculate the rest of the new matrix using the Montante formula.
  5. Repeat the process with the next pivot.
  6. Read the final solution from the diagonal matrix.

Special cases

Infinitely many solutions

If a complete zero row appears, it means at least one equation depends on the others. In that case, the system has free variables and therefore infinitely many solutions.

0x1 + 0x2 + ... + 0xn = 0

No solution

If the coefficients in a row are all zero but the constant is not zero, the system contains a contradiction and has no solution.

0x1 + 0x2 + ... + 0xn = c, where c ≠ 0

Example

Consider the following system:

x1 + x2 + x3 = 4

This equation alone does not determine a unique value for all variables. If we let x2 = t and x3 = s, then:

x1 = 4 - t - s
x2 = t
x3 = s

In vector form:

X = [4, 0, 0] + t[-1, 1, 0] + s[-1, 0, 1]
