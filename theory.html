Bareiss-Montante Method

A short theoretical guide for solving systems of linear equations.

Calculator Theory

What is the Bareiss-Montante method?

The Bareiss-Montante method is an elimination method used to solve systems of linear equations through an augmented matrix. Its objective is similar to Gauss-Jordan reduction: to transform the system into a form where the solution can be read clearly.

One advantage of this method is that it usually preserves integer values during the process, which makes the calculations cleaner. The method advances through a fixed number of stages based on the number of rows in the matrix.

Main formula

At each stage, the method uses the current pivot and the previous pivot to calculate the new values of the matrix.

a'ij = (pivot · aij - aik · akj) / previous pivot

In the first stage, the previous pivot is equal to 1. After that, the previous pivot becomes the pivot from the stage before.

Augmented matrix

A system with n variables can be represented using an augmented matrix with n rows and n + 1 columns. The last column contains the constants from the right side of the equations.

x + y + z = 6
2x - y + z = 3
x + 2y - z = 2

This system is represented as:

[ 1   1   1 | 6 ]
[ 2  -1   1 | 3 ]
[ 1   2  -1 | 2 ]

Step-by-step procedure

  1. Select the pivot from the main diagonal. The pivot must be different from zero.
  2. Keep the pivot row unchanged.
  3. Turn the other values in the pivot column into zero.
  4. Calculate the rest of the new matrix using the Bareiss-Montante formula.
  5. Repeat the process with the next pivot.
  6. At the end, divide each row by its diagonal value to obtain the identity matrix.

Example

Consider the following augmented matrix:

[ 1   1   1 | 6 ]
[ 2  -1   1 | 3 ]
[ 1   2  -1 | 2 ]

In the first stage, the pivot is 1 and the previous pivot is also 1. The pivot row is copied into the new matrix, and the other values in the pivot column become zero.

For example, to calculate the value in position a'22:

a'22 = ((1)(-1) - (2)(1)) / 1 = -3

The same formula is applied to the other positions that are not in the pivot row or pivot column. After repeating the process through the matrix, the method produces a diagonal matrix.

[ 7   0   0 | 7 ]
[ 0   7   0 | 14 ]
[ 0   0   7 | 21 ]

Finally, each row is divided by its diagonal value:

[ 1   0   0 | 1 ]
[ 0   1   0 | 2 ]
[ 0   0   1 | 3 ]

Therefore, the solution is:

x1 = 1
x2 = 2
x3 = 3

Special cases

Infinitely many solutions

If a complete zero row appears, it means that at least one equation depends on the others. In this case, the system has free variables and therefore has infinitely many solutions.

0x1 + 0x2 + ... + 0xn = 0

No solution

If the coefficients in a row are all zero but the augmented value is not zero, the system contains a contradiction.

0x1 + 0x2 + ... + 0xn = c, where c ≠ 0

This is impossible because the left side is always zero, so the system is inconsistent and has no solution.
